# 🏋️ FitBook - 운동 기록 웹앱

운동 기록을 간편하게 저장하고 관리할 수 있는 웹 애플리케이션입니다.  
Node.js 기반 API와 프론트엔드를 연동하여 실제 동작하는 웹앱 형태로 구현했습니다.

---

## 📌 프로젝트 개요

FitBook은 사용자가 자신의 운동 기록을 쉽게 추가, 조회, 수정, 삭제할 수 있도록 만든 웹 서비스입니다.

단순 CRUD 구현을 넘어서,  
👉 사용자 경험(UI/UX)과 실제 동작 흐름을 고려하여 설계했습니다.

---

## 🚀 주요 기능

- ✅ 운동 기록 추가
- ✅ 운동 기록 조회
- ✅ 운동 기록 수정
- ✅ 운동 기록 삭제
- ✅ 총 운동 시간 및 개수 표시
- ✅ 날짜 기반 기록 관리

---

## 🛠 기술 스택

- **Backend**: Node.js, Express
- **Frontend**: HTML, CSS, JavaScript
- **데이터 저장**: 메모리 기반 (추후 DB 확장 가능)

---

## 💡 구현 포인트

- REST API 구조 설계 (GET, POST, PUT, DELETE)
- 프론트엔드와 API 연동
- 사용자 중심 UI (카드형 디자인 + 버튼 인터랙션)
- 실시간 데이터 반영 (새로고침 없이 동작)
